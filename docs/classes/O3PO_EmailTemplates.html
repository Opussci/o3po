self_notification_subject(string  $template, string  $journal, string  $publication_type_name) : Mixed

Replace the short codes in the self notification subject template.

Parameters

string $template

The template with the self notification subject.
string $journal

The journal name.
string $publication_type_name

The type of the publication.

Returns

Mixed —

The function returns a map with two keys. result: The template in which the short codes were replaced. short_codes: A map where the keys are the short codes and the values are the descriptions of the short codes.
